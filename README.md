<h2>Lenguaje de programación<h2/>

<h4>Este debe ser formal y claro este es el que nos permite escribir las instrucciones para crear un algoritmo, esto con el fin de controlar el comportamiento físico o ya sea lógico de una computadora. A este lenguaje y las ordenes que nos permite dar es lo que conocemos como programa.<h4/>

<img width="642" height="350" alt="image" src="https://github.com/user-attachments/assets/99f71b2b-7db2-4025-8a19-7a1c5ef9ea97" /> Imagen 2


<h2>Tipos de lenguajes<h2/>
<h4></h4>Interpretados (traducidas una a una): Python, Perl, Bash.
Compilados (traducidas en grupo): Fortran, C, C++.
Intermedios (parte compilados, parte interpretados): Java, Scala<h4/>



<h2>Nivel de clasificación:<h2/>
<h4></h4>Alto: información compleja, objetos simples: Python, C++, C#, Java.
Bajo: información compleja se representa literalmente: Assembly y Machine.<h4/>


<h2>Entornos de desarrollo:<h2/>
</h4>También se les conoce como IDE (Integrated Development Enviroment) son editores de código que se diseñan para que nosotros los desarrolladores escribamos código o ayudarnos a testear para ver su ejecución si es buena o incorrecta. Este también nos permite tener más control sobre los recursos de un sistema<h4/>
